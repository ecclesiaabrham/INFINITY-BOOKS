# Infinite Books

The purpose of this website is to provide a simple way to browse and search for books by category or topic.

## Features

* Browse available books
* Search for books by category or keyword
* View book cover images
* Open books in the browser
* Responsive grid layout
* Dark mode
* Description/about page
* Random book recommendations on the home page

## Technologies Used

* HTML
* CSS
* JavaScript

## About the Project

This is my first web development project. I built it to practice HTML, CSS, and JavaScript, including DOM manipulation, event listeners, search functionality, dynamic content, and local storage.
